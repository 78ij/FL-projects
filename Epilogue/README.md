# Epilogue

## 流派

Neo-Classical (?)

##  创作时间

2017.10

## 评论

和《Intro》相似的曲子，但是一开始没有考虑配对问题，不知不觉就变成这样了（笑

不过和《Intro》相比，采用了g小调这么一个具有魔力的调性，并且加入了中提琴、铜管。

也许会给人一种结束的感觉吧。
